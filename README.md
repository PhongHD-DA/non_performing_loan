# Employee Workforce Analytics Dashboard

This project analyzes workforce performance, employee retention, satisfaction, and business impact using HR and operational data from 2022–2024. The goal is to transform raw employee data into actionable business insights through SQL analytics and interactive Power BI dashboards.

---

# Project Overview

The project focuses on answering key workforce and organizational questions such as:

- Which departments have the highest turnover rates?
- Are highly paid employees also high performers?
- Does employee satisfaction correlate with performance and retention?
- Which workforce segments create the highest business value?
- How does workforce structure impact operational efficiency?

The analysis combines HR metrics, employee performance, satisfaction, compensation, and sales-related operational data to support data-driven workforce decisions.

---

# Data Processing & Analytics

## SQL Analytics Layer

Raw employee and operational datasets were cleaned, transformed, and analyzed using SQL.

Key analytical tasks included:

- Employee turnover analysis
- Compensation and performance segmentation
- Workforce productivity analysis
- Department-level retention risk analysis
- Satisfaction and performance correlation
- Workforce effectiveness measurement

---

# Power BI Dashboard

Power BI was used to build an interactive executive-style analytics dashboard with business storytelling structure.

## Dashboard Structure

### 1. HR Overview
Executive-level summary of:
- Total sales
- Workforce headcount
- Sales achievement
- Turnover rate
- Sales and workforce trends

### 2. Workforce Insights
Analysis of:
- Compensation vs performance
- Workforce segmentation
- Job-level performance
- High-risk and high-potential employees

### 3. Workforce Health
Focuses on:
- Employee satisfaction
- Workforce stability
- Retention risk analysis
- Department-level workforce health

### 4. Workforce Impact
Measures:
- Productivity by department
- Workforce efficiency
- Compensation effectiveness
- Relationship between workforce quality and business outcomes
